# Paradise Nursery Shopping Application

Paradise Nursery is a React and Redux Toolkit shopping cart application for an online plant shop.

## Features
- Landing page with the Paradise Nursery name and Get Started button.
- About Us page with company details.
- Product listing page with 3 plant categories.
- At least 6 unique houseplants in each category.
- Add to Cart buttons that disable after adding an item.
- Dynamic cart count in the navigation bar.
- Shopping cart page with quantity increase, decrease, delete, total cost, checkout, and continue shopping.

## Run
npm install
npm run dev
